Principais Tags do HTML

<HTML>

A primeira coisa que você ter em mente é que o HTML é uma linguagem única e exclusivamente para a semântica, ou seja, para dar significado aos elementos.

<h>

A tag <h> serve para definir a hierarquia de um texto, por exemplo:

  • <h1>
    • <h2>
      • <h3>
        • <h4>
          • <h5>
            • <h6>

<p>

A tag <p> define um parágrafo.

<ul>

A tag <ul> define uma lista desordenada.

Exemplo:

  • Item 1 da lista.
  • Item 2 da lista.
  • Item 3 da lista.
  • E assim por diante...

<ol>

A tag <ol> define uma lista ordenada.

Exemplo:

  1. Item 1 da lista.
  2. Item 2 da lista.
  3. Item 3 da lista.
  4. E assim por diante...

<i> e <em>

As tags <i> e <em> podem parecer fazer a mesma coisa, porém tem papéis diferentes. Enquanto a tag <i> define o itálico para auxiliar a leitura do usuário, a tag <em> define o itálico para auxiliar o navegador. Ou seja, a tag <i> é visual, enquanto a tag <em> é semântico.

Exemplo:

  • Esta palavra em itálico ajuda a leitura do usuário.
  • Enquanto essa palavra em itálico ajuda o navegador.

<b> e <strong>

Da mesma forma são as tags b e strong, enquanto a i define de forma visual, a strong define de forma semântica.

Exemplo:

  • Esta palavra em negrito ajuda a leitura do usuário.
  • Enquanto esta palavra em negrito ajuda o navegador.

<a>

A tag <a> serve para a ancoragem, para linkar uma página com a outra, tanto interna quanto externa. Ela pode ser usada com alguns atributos.

HTML 

    <a href="..." target="...">

<br> e <hr>

As tags <br> e <hr> tem seus papéis muito bem definidos, enquanto uma faz quebra de linha, a outra faz uma linha na horizontal da página, respectivamente.

<mark>

A tag <mark> pode parecer inútil a primeiro momento, pois tem a função de marcar uma palavra, ou texto, e o papel visual já é dado ao CSS. Porém, pensando de forma semântica, ela será usada para informar ao navegador que alí tem algo que merece atenção do usuário.

Exemplo:

  • Esta palavra está marcada com a tag <mark>.

<img>

A tag <img> é usada para exibir uma imagem em uma página HTML utilizndo os atributos src="..." (para o caminho da imagem) alt="..." (para texto alternativo) como os únicos obrigatórios. Essa tag é um elemento vázio, o que significa que não precisa de uma tag de fechamento.

HTML 

    <img src="..." alt="...">
