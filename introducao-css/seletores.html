Seletores

class

O atributo class="...", refere-se a uma classe de tags, possibilitando a personalização de um conjunto de elementos.

HTML

    <p class="my-class">...</p>
CSS
CSS 

    .my-class {

    }

font

font-family

Formata para a família da fonte.

CSS 

    p {
        font-family: 'Times New Roman', Times, serif;
    }
font-size

Redefine o tamanho da fonte.

CSS 

    p {
        font-size: 0.6em;
    }
font-style

Redefine o estilo da fonte, itálico por exemplo.

CSS 

    p{
        font-style: italic;
    }
font-weigth

Define negrito para os textos.

CSS 

    p {
        font-weight: bold;
    }

text

text-align

Para alinhar os textos à esquerda, no centro ou na direita.

CSS 

    p {
        text-align: left;
    }
text-decoration

Esse atributo altera alguns aspectos do texto, por exemplo:

  • none: remove qualquer decoração padrão.
  • underline: adicione uma linha abaixo do texto.
  • line-through: adiciona uma linha que passa pelo meio do texto.
  • overline: adiciona uma linha acima do texto.
CSS 

    p {
        text-decoration: underline;
    }
text-ident

Define identação no início de cada parágrafo.

CSS 

    p {
        text-ident: 1.5em;
    }
text-transform

Esse atributo tem a função de transformar alguns aspectos dos textos.

  • uppercase: permite transformar o texto em maiúsculo.
  • lowercase: permite transformar o texto em minúsculo.
  • capitalize: converte a primeira letra de cada palavra para maiúsculo.
  • none: remove qualquer característica padrão do atributo.
CSS 

    p {
        text-transform: uppercase;
    }

color

Define a cor do texto.

CSS 

    p {
        color: #9acd32;
    }

background-color

Define a cor de fundo do elemento.

CSS 

    p {
        background-color: #9acd32;
    }
