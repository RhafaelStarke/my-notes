Fit-content

max-content

O max-content diz ao navegador: "Largura total! Não quebre a linha de jeito nenhum, a menos que eu peça com <br>".

  • Comportamento: a caixa vai se alargar até que todo o conteúdo caiba em uma única linha.
  • Perigo: se o texto for muito longo, ele vai vazar para fora da tela e criar uma barra de rolage, horizontal.
  • Uso: bom para ícones com labels curtos ou quando você quer garantir que um título nunca quebre.
CSS 

.exemple-max-content {
    width: max-content;
}

min-content

O min-content diz ao navegador: "Fique o menor possível, apertando tudo o que der.".

  • Comportamento: a caixa vai encolher até a largura da maior palavra (ou do elemento inquebrável, como uma imagem) dentro dela. O texto vai quebrar em várias linhas o máximo que puder.
  • Uso: útil para legendas de imagens ou colunas de tabelas que devem ser estreitas.
CSS 

.exemple-min-content {
    width: min-content;
}

fit-content

O fit-content é o melhor dos dois mundos. Ele diz: "Use o espaço que o conteúdo precisa (max-content), mas se o espaço acabar, comece a quebrar a linha."

  • Comportamento:
    • Se o texto é curto, ele age como max-content.
    • Se o texto é longo, ele cresce até atingir o limite do elemento pai.
    • A partir desse limite, ele para de crescer e quebra o texto (age como auto).
  • Uso: perfeito para centralizar blocos de conteúdo, botões ou tooltips.
CSS 

.exemple-fit-content {
    width: fit-content;
}

over-flow

A propriedade over-flow pode assumir quatro comportamentos diferentes.

over-flow: visible (O Padrão)
  • O que faz: o conteúdo transborda e vaza para a fora da caixa, ficando visível por cima de outros elementos.
  • Resultado: é bagunçado. O texto sai da borda.
  • Quando usar: geralmente é o comportamento que queremos evitar em caixas de texto, mas é útil quando você quer que algo saia da caixa.
CSS 

.over-flow {
    overflow: visible;
}

over-flow: hidden
  • O que faz: o conteúdo que não cabe é simplesmente cortado, ou escondido. O resto do conteúdo fica invisível e inacessível.
  • Resultado: a caixa fica limpa, respeitando o tamanho definido.
  • Quando usar:
    • Para criar efeitos de imagem (arredondar cantos de uma imagem dentro de uma div).
    • Para esconder animações que saem da tela.
    • Antigamente, era usado para corrigir problemas de float.
CSS 

.over-flow-hidden {
    overflow: hidden;
}

over-flow: scrool
  • O que faz: adiciona barras de rolagem (scrollbars) na caixa, tanto na horizontal quanto na vertical.
  • Detalhe importante: as barras aparecem mesmo se não forem necessárias.
  • Quando usar: raramente. Geeralmente preferimos o auto.
CSS 

.over-flow-scroll {
    overflow: scroll;
}

over-flow: auto
  • O que faz: adiciona barras de rolagem apenas se forem necessárias.
  • Resultado: se o conteúdo couber, parece uma caixa normal. Se o conteúdo crescer, a barra de rolagem aparece automaticamente.
  • Quando usar: é a melhor opção para áreas de textos com rolagem, como blocos de códigos
CSS 

.over-flow-auto {
    overflow: auto;
}
