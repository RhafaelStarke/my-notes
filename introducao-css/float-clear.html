Float e Clear

float (Flutuar)

A propriedade float tira o elemento do fluxo normal da página e o "empurra" para um dos lados (esquerda ou direita), permitindo que o texto e outros elementos de linha "flutuam" ao redor dele.

Valores:

  • float: left;: flutua à esquerda.
  • float: right;: flutua à direita.
  • float: none;: valor padrão (não flutua).

Exemplo:

CSS 

img {
    float: right; /*fixa a imagem à direita*/
    margin-left: 15px; /*define uma margem para o texto não ficar colado*/
    width: 200px /*define a largura da imagem*/
}

Atenção

Quando você usa float, você cria um "efeito colateral". O elemento flutuante sai do fluxo normal. Isso causa dois problemas comuns:

  1. O elemento seguinte tenta subir: se você tem uma imagem flutuando e depois um título ou rodapé que deveria estar embaixo, esse elemento pode tentar subir para o espaço ao lado da imagem, assim como fez o texto, quebrando o layout.
  2. O pai colapsa: Se uma div contém apénas elementos flutuantes, a altura dela vira 0. A borda e o fundo da div somem, porque ela acha que está vazia (já que os filhos estão flutuando).

clear (Limpar)

A propriedade clear serve para dizer ao navegador: "Pare de flutuar aqui! Quero começar uma nova linha limpa, abaixo de qualquer coisa que esteja flutuando."

Valores:

  • clear: left; não permite elementos flutuar à esquerda.
  • clear: right; não permite elementos flutuar à direita.
  • clear: both; o mais usado, não permite nada flutuando em nenhum dos lados.

Exemplo:

CSS 

.imagem-lateral {
    float: left;
}

.rodape {
    clear: both; /*garante que o rodapé fique embaixo da imagem*/
}

Clearfix

Lembra que eu disse que o container pai não reconhece seus filhos flutuantes, e fica com altura zero?

Antigamente, usávamos um "hack" chamado clearfix para resolver isso. Hoje existe uma solução moderna de uma linha só.

Se você tiver uma div com elementos flutuantes dentro, aplique essa propriedade:

CSS 

.container-pai {
    display: flow-root;
}

Isso força o pai a reconhecer a altura dos filhos flutuantes.
