Images Vetoriais vs Bitmap

Imagens bitmap são formadas por pixels, enquanto imagens vetoriais são definidas por fórmulas matemáticas, usando formas geométricas, linhas e curvas. Como resultado disso, bitmap está limitado à sua definição padrão, enquanto imagens vetoriais se adequam ao tamanho que for necessário, sem abrir mão da definição.

Float

O float em CSS fazem a imagem sair do fluxo normal do texto e se posicionar ao lado esquerdo ou direito do seu contêiner, permitindo que outros elementos, como o texto, contornem e se ajustem ao redor dela.

Exemplo:

HTML 

    <img src="sua-imagem.jpg" alt="descrição da imagem" class="float-image">

CSS 

    .float-image {
        float: left;
        margin 0 10px 10px 0;
    }

background-image

Pense na background-image como uma verdadeira tela de pintura, onde você pode adicionar camadas, gradientes, texturas e efeitos.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
    }

background-repeat

  • repeat: a imagem irá se repetir.
  • no-repeat: a imagem não irá se repetir.
  • repeat-x: a imagem irá se repetir somente no eixo x.
  • repeat-y: a imagem irá se repetir somente no eixo y.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: repeat; /*padrão*/
        background-repeat: no-repeat;
        background-repeat: repeat-x;
        background-repeat: repeat-y;
    }

background-size

Controla o tamanho da imagem com as seguintes propriedades.

auto

Faz com que a imagem de fundo mantenha seu tamanho original, sem esticar ou cortar. É o valor padrão da propriedade.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: auto;
    }
cover

Redimensiona uma imagem de fundo para cobrir completamente seu contêiner, preservando a proporção original da imagem, mesmo que isso signifique que parte da imagem seja cortada. O objetivo é não deixar nenhum espaço vazio e garantir que o fundo sempre preencha toda a área, ajustando-se dinamicamente às proporpoções do contêiner.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: cover;
    }
contain

Redimensiona uma imagem de fundo para o maior tamanho possível, mantendo suas proporções originais, garantindo que ela caiba completamente dentro da área do elemento. Isso significa que, dependendo das proporções da imagem e do elemento, podem aparecer áreas vazias ao redor da imagem, pois ela é redimensionada para não ser cortada.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: contain;
    }
50%

Redimensiona a imagem de fundo para 50% da largura do elemento pai e, como apenas um valor valor foi especificado, a altura é automaticamente ajustada para manter a proporção da imagem original, evitando distorções.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: 50%;
    }
100px

Define que a imagem de fundo do elemento terá a largura de 100 pixels, e a altura será definida automaticamente para manter a proporção original da imagem.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: 100px;
    }

background-position

A propriedade background-position define a posição inicial de uma imagem de fundo de um elemento HTML, permitindo que ela seja movida para diferentes locais usando valores de palavras chaves (como top, right, bottom, left, center), porcentagens, ou unidades de medida como pixels. Você especifica a posição horizontal (eixo X) e a posição vertical (eixo Y), onde o primeiro valor é para o eixo X e o segundo o eixo Y, como em background-position: 50% 50% para centralizar a imagem.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: cover;
        background-position: right bottom;
    }
background-attachment

A propriedade background-attachment no CSS define se a imagem de fundo de um elemento permanecerá fixa em relação à tela (viewport) ou se rolará junto com o cconteúdo. Os valores principais são:

  • scroll: a imagem rola com o conteúdo, sendo padrão.
  • fixed: a imagem fica fixa na tela mesmo com a rolagem.
  • local a imagem rola com o conteúdo do próprio elemento, mas se comporta de forma diferente de scroll em elementos com overflow.

Exemplo:

CSS 

    body {
        background-image: url(../img/bg.jpg);
        background-repeat: no-repeat;
        background-size: cover;
        background-position: right bottom;
        background-attachment: fixed;
    }
