Pseudo Classes

Pseudo-classes nos permite estilizar elementos com base em seu estado ou condição, e não apenas em sua identidade, como tag classe ou id.

<a>

Um bom exemplo para isso é a tag <a>, um link pode estar em diferentes estados durante a interação do usuário.

:link

Representa o estado de um link que ainda não foi visitado pelo usuário.

CSS 

    .a:link {
        color: #0000ff;
        text-decoration: none;
    }

Resultado:

  • Exemplo de um link.
:visited

Representa o estado de um link que já foi visitado.

CSS 

    .a:visited {
        color: #6a0dad;
    }

Resultado:

  • Exemplo de um link.
:hover

Representa o estado quando o usuário passa o cursor do mouse sobre o link.

CSS 

    .a:hover {
        color: #0800ff;
        text-decoration: underline;
    }

Resultado:

  • Exemplo de um link.
:active

Representa o estado quando o link está sendo ativamente clicado, no momento em que o botão do mouse é pressionado até ser solto.

CSS 

    .a:active {
        color: #ffd700;
    }

Resultado:

  • Exemplo de um link.

A Ordem

Quando você define os estilos para os estados de um link, a ordem em que você escreve as regras no seu arquivo CSS é crucial. Se a ordem estiver errada, alguns estilos não funcionarão como esperado por causa das regras de especificidade do CSS.

A ordem é:

  1. :link
  2. :visited
  3. :hover
  4. :active

Outras Pseudo-classes Essenciais

O conceito de pseudo-classes se aplica a quase todos os elementos, não apenas a de links. Algumas outras muito importantes são:

  • :focus: aplica um estilo quando um elemento está em foco (selecionado via teclado com a tecla tab).
  • :first-child: seleciona um elemento um elemento que é o primeiro filho de seu pai.
  • :last-child: seleciona um elemento que é o último filho do seu pai.
  • :nth-child(n): seleciona elementos com base em sua posição. Ex: nth-child(even) seleciona os filhos pares (ótimo para zebrar tabelas).
