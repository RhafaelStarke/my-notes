margin

É o espaçamento externo da caixa. Ele cria um espaço entre a sua caixa e os elementos vizinhos. A margem é sempre transparente, empurrando os outros elementos para longe.

    #exemple-margin {
        width: 250px;
        height: 25px;
        background-color: #9acd32;
        padding: 25px;
        border: 10px solid #aaff00;

outline

O outline é uma linha desenhada fora da borda. Ele é muito parecido com o border, mas tem sua diferença crucial: O outline não ocupa espaço, ele não afeta o layout da página e não empurra os outros elementos. Ele é desenhado "por cima" do espaço da margin.

CSS 

    #exemple-outline {
        width: 250px;
        height: 25px;
        background-color: #9acd32;
        padding: 25px;
        border: 10px solid #aaff00;
        margin: auto;
        outline: 5px dashed #ff0000;
        outline-offset: 5px;
    }

box-sizing

Por padrão no CSS, quando você define o tamanho do width, o padding e o border são somados a esse valor.

  • Largura total = width + padding-left + padding-right + border-left + border-right

Portanto, a melhor prática é utilizar border-box. Com ele, o padding e o border são incluídos dentro da largur que você definiu.

Se você definiu a largura (width) com 200 píxels, a caixa inteira, incluindo padding e o border terá o valor defido da largura. O próprio CSS ajusta o espaço do conteúdo que caiba tudo.

CSS 

    #exemple-box-sizing {
        width: 250px;
        height: 25px;
        background-color: #9acd32;
        padding: 25px;
        border: 10px solid #aaff00;
        margin: auto;
        outline: 5px dashed #ff0000;
        box-sizing: border-box;
    }

block

Alguns elementos, como <p>, <h>, <ul>, <ol>, <li> e outros são, por padrão, elementos block. Isso significa que vão ocupar toda a largura do elemento pai, e vão gerar quebra de linha.

inline

Já outros, como <a>, <b>, <strong>, <i>, <em> e outros, são, por padrão, elementos inline. Isso significa que não gerar quebra de linha e irão ocupar apenas o espaço necessários para cabê-los.

display

Utilizando da propriedade display você consegue alterar o valor padrão do elemento.

  • inline: altera os elementos em blocos para elementos em linha.
  • block: altera os elementos em linha para elementos em blocos.
  • inline-block: faz com que elememtos em linha possam receber características de em bloco (padding e margin, por exemplo).
  • none: remove o elemento da tela.
